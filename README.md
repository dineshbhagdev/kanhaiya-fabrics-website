# Kanaiya Fabrics Static Website

This folder contains a static multi-page website for `https://kanaiyafabrics.in/`.

## Files

- `index.html` - home page
- `about.html` - business story and values
- `products.html` - product category page
- `reviews.html` - customer review highlights
- `contact.html` - contact and visit details
- `styles.css` - responsive layout and visual styling
- `script.js` - mobile menu behavior shared by all pages
- `assets/` - generated website imagery used by the page

## Local Preview

Run this from the site folder:

```sh
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173/
```

## Publish

Upload the contents of this folder to the domain's web root on the current hosting provider. For GoDaddy Website Builder, this usually means replacing the builder site with custom hosting or uploading these files through cPanel/File Manager if hosting is enabled.
